# Subnetting using VLSM (Variable Length Subnet Masking)

## 📌 Overview

This project demonstrates subnetting using Variable Length Subnet Masking (VLSM) and communication between multiple subnets using static routing in Cisco Packet Tracer.

## 🎯 Objective

* Divide a network into multiple subnetworks using VLSM
* Efficiently allocate IP addresses
* Configure routers for communication between subnets
* Verify connectivity using ping

## 🛠 Tools Used

* Cisco Packet Tracer

## 🌐 Given Network

14.24.74.0/24

## 📂 Subnet Requirements

* 120 hosts
* 60 hosts
* 10 hosts

## ⚙️ Subnets Created

| Hosts Required | Subnet Assigned |
| -------------- | --------------- |
| 120 Hosts      | 14.24.74.0/25   |
| 60 Hosts       | 14.24.74.128/26 |
| 10 Hosts       | 14.24.74.192/28 |

## ✅ Verification

* Successful communication between subnets
* Verified using Simple PDU and ping

## 📚 Learning Outcome

* Understanding VLSM
* Efficient IP allocation
* Static routing between subnetworks
