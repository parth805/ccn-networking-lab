## 🔍 Detailed Explanation

### Step 1: Network Creation

* Added 1 router, 1 switch, and 2 PCs
* Connected devices using copper straight-through cables

### Step 2: IP Addressing

Assigned IP addresses manually:

* PC0 → 192.168.1.2
* PC1 → 192.168.1.3
* Default Gateway → 192.168.1.1

### Step 3: Router Configuration

Accessed CLI and executed:

enable
configure terminal
interface gigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown

### Step 4: Testing

* Used command prompt on PC
* Executed: ping 192.168.1.1

### 📊 Result

Ping was successful, confirming proper network communication.

### 💡 Key Concepts

* Router acts as gateway
* IP addressing enables identification of devices
* Ping verifies connectivity

### 🧠 Insight

Even a simple LAN setup requires proper gateway configuration. Without router configuration, communication fails.
