# Router Configuration & LAN Communication

## 📌 Overview

This project demonstrates how to configure a router in Cisco Packet Tracer to enable communication between devices within a Local Area Network (LAN).

## 🎯 Objective

* Configure router interfaces
* Assign IP addresses to end devices
* Enable communication between devices via a router

## 🛠 Tools Used

* Cisco Packet Tracer

## 🌐 Network Topology

* 1 Router
* 1 Switch
* 2 PCs

(All devices connected using copper straight-through cables)

## ⚙️ Key Configuration

* Router Interface: GigabitEthernet0/0
* IP Address: 192.168.1.1
* Subnet Mask: 255.255.255.0

## ✅ Verification

* Successful ping from PC to router
* Network connectivity established

## 📚 Learning Outcome

* Understanding of basic router configuration
* Role of default gateway in communication
* Hands-on experience with Packet Tracer
