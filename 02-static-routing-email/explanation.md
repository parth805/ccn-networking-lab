## 🔍 Detailed Explanation

### Step 1: Network Setup

* Created 3 LANs connected via 3 routers
* Each LAN assigned different IP classes (A, B, C)
* Devices connected using switches

### Step 2: IP Addressing

* Assigned IP addresses to all PCs and routers
* Configured default gateways

### Step 3: Static Routing

* Manually configured routes in each router
* Ensured all networks are reachable

### Step 4: Verification

* Used ping command to test connectivity
* Used Simple PDU to visualize packet flow

### Step 5: Email Configuration

* Enabled email service on server
* Created user accounts
* Configured email on PCs

### Step 6: Email Testing

* Sent email from one LAN to another
* Verified successful delivery and reply

## 📊 Result

* Successful communication between different networks
* Email services working across LANs

## 💡 Key Concepts

* Static Routing
* IP Addressing
* SMTP Protocol
* Inter-network communication

## 🧠 Insight

Static routing is simple and effective for small networks but requires manual configuration, making it less scalable for larger systems.
