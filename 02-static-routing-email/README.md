# Static Routing & Email Services in LAN

## 📌 Overview

This project demonstrates the implementation of static routing between multiple LANs and the configuration of email services using Cisco Packet Tracer.

## 🎯 Objective

* Configure multiple LANs with different IP classes
* Implement static routing between routers
* Enable communication across networks
* Configure and test email services

## 🛠 Tools Used

* Cisco Packet Tracer

## 🌐 Network Design

* 3 Routers
* 3 Switches
* Multiple PCs
* 1 Server

Each LAN uses a different IP class (A, B, C).

## ⚙️ Key Features

* Manual route configuration (static routing)
* Inter-network communication using routers
* Email service using SMTP protocol
* Verification using ping and Simple PDU

## ✅ Verification

* Successful ping between different LANs
* Email sent and received across networks

## 📚 Learning Outcome

* Understanding of static routing
* Configuration of multi-network topology
* Real-world application of network services
