# RIP Routing using Cisco Packet Tracer

## 📌 Overview

This project demonstrates the implementation of Routing Information Protocol (RIP) to enable dynamic routing between multiple LAN networks using Cisco Packet Tracer.

## 🎯 Objective

* Configure routers using RIP
* Establish communication between multiple LANs
* Enable automatic route exchange between routers
* Verify connectivity using ping

## 🛠 Tools Used

* Cisco Packet Tracer

## 🌐 Network Design

* 3 Routers
* 3 Switches
* Multiple PCs
* Different Class A, B, and C Networks

## ⚙️ Key Features

* Dynamic Routing using RIP
* Automatic routing table updates
* Inter-network communication
* Packet verification using Simple PDU

## ✅ Verification

* Successful ping between different LANs
* Routing tables updated automatically

## 📚 Learning Outcome

* Understanding dynamic routing
* Configuration of RIP protocol
* Difference between static and dynamic routing
